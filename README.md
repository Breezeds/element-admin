# element-admin

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 安装 axios
```
cnpm i axios
```

### 服务端 server 安装
```
cnpm i express@next mongoose cors
```
