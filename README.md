# element-admin

### 项目简介
使用vue写前端页面，使用expres实现后台服务，使用mongoDB管理数据，实现帖子的增删改查功能

### 技术栈
Vue
express
mongoose
axois
cors

### Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 安装 axios
```
cnpm i axios
```

### 服务端 server 安装
```
cnpm i express@next mongoose cors
```
